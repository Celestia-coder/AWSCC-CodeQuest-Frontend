Inline CSS

Inline CSS is a way to apply styles directly to individual HTML elements on a web page.

Internal CSS

Internal CSS is when you put your styles in the style section within the HTML document.

External CSS

External CSS is where you create a separate CSS file and link it to your HTML document using the < link> element.

CSS Selector

CSS selectors streamline the process of styling HTML elements by enabling you to target multiple elements simultaneously.
