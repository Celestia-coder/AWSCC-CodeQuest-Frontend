NON-SEMANTIC ELEMENTS

Div element
Use < div >
This element is a block-level container used to group and structure content together.
Span element
Use < div >
This element is an inline-level container used to apply styles or scripts to small portions of text or inline elements.

We can also change color text using < span style="color: color;">

We can also highlight text using < span style="background-color: color;">

We can also add font style using < span style="font-style: italic; font-weight: bold;">

We can also underline text using < span style="text-decoration: underline;">

We can also Strikethrough text using < span style="text-decoration: line-through;">

Lastly, just combine everything!


SEMANTIC ELEMENTS

This is header tag

The < header > element represents the introductory content or a group of navigational links at the beginning of a document or section. It typically contains headings, logos, and navigation menus.

This is nav tag

The < nav> element is used to define a navigation menu on a web page. It often contains an unordered list < ul> of links < a> to different sections or pages of the website.

This is main tag

The < main> element represents the main content of a document, excluding headers, footers, and sidebars.

This is article tag

The < article> element encapsulates content that can stand alone and be distributed and reused independently. It could be a blog post, a news article, a forum post, or any piece of content with its context.


This is section tag

The < section> element groups related content together and typically includes a heading (< h1> to < h6>). It's used to organize content into distinct sections, enhancing the document structure.


This is aside tag

The < aside> element represents content that is tangentially related to the content around it. It's often used for sidebars, pull quotes, or other content that complements the main content.


This is footer

The < footer> element represents the footer of a section or a page. It typically contains metadata, copyright information, links to related documents, and other supplementary information.
