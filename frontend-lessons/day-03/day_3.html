NON-SEMANTIC ELEMENTS

Div element
Use < div >
This element is a block-level container used to group and structure content together.
Span element
Use < div >
This element is an inline-level container used to apply styles or scripts to small portions of text or inline elements.

We can also change color text using < span style="color: color;">

We can also highlight text using < span style="background-color: color;">

We can also add font style using < span style="font-style: italic; font-weight: bold;">

We can also underline text using < span style="text-decoration: underline;">

We can also Strikethrough text using < span style="text-decoration: line-through;">

Lastly, just combine everything!
